Caesar Cipher Encoder/Decoder

How to Use

The Caesar Cipher is one of the simplest and most widely known encryption techniques. It works by shifting each letter in the plaintext by a fixed number of positions down the alphabet.

Encoding

To encode a message:

  1. Enter your message in the input field
  2. Choose a key between 0 and 25
  3. Select "Encode"
  4. The encoded message will appear in the output field

Decoding

To decode a message:

  1. Enter the encoded message in the input field
  2. Enter the same key that was used for encoding
  3. Select "Decode"
  4. The original message will appear in the output field

Example

With a key of 3:

  • "HELLO" encodes to "KHOOR"
  • "KHOOR" decodes to "HELLO"
